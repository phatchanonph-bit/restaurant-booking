# Frontend Structure

- `index.html`: หน้าแรกของร้าน
- `booking.html`: ฟอร์มกรอกข้อมูลการจอง
- `select-seat.html`: หน้าเลือกโต๊ะ
- `success.html`: หน้ายืนยันการจองสำเร็จ
- `admin-login.html`: หน้าเข้าสู่ระบบแอดมิน
- `admin.html`: หน้าแอดมินจัดการรายการจอง
- `css/`: รวมไฟล์สไตล์ของแต่ละหน้า
- `js/`: รวมไฟล์สคริปต์ของแต่ละหน้า

หมายเหตุ:
- ถ้าจะแก้หน้าตา ให้เข้าไปที่ `css/`
- ถ้าจะแก้ logic หน้าเว็บ ให้เข้าไปที่ `js/`
